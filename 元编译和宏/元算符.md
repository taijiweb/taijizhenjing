元算符

太极语言提供了一组元算符，包括#，##，#/，#&，#/=，#&=，#-等。本章将解释这些算符的功能和用法。

元算符使用示例

#(1+1)

# (#(1+2) + #(3+4))

#(1+2) + #(3+4)

3+.#(1+1)

# ~ 1+1

##a=1

a#=1

#(a=1)

a#=1;#a

## if 1 then 1 else 2

if 1 then #1+2 else #3+4

if 1 then ##1+2 else ##3+4

compileNoOptimize if 1 then ##1+2 else ##3+4

array? #= (obj) -> ` (Object::toString.call(^obj) == '[object Array]')

array? # 1

array? # []

#  #-{ print 1 } 编译结果是 `console.log(1)`

` #{ -> #- { print 1 }})() ` 编译结果是 `[print, 1]`
